# InvestmentBankingAI

## Overview

InvestmentBankingAI is an AI-powered Equity Research & Valuation Platform that automatically generates institutional-style equity research reports for NSE-listed companies.

The platform combines live financial data, valuation models, AI-generated analysis, and PDF automation into a single workflow.

---

## Features

- Live Financial Data (Yahoo Finance)
- DCF Valuation
- Peter Lynch Valuation
- Dynamic Buy/Hold/Sell Recommendation
- Financial Health Scorecard
- Investment Scorecard
- Peer Comparison
- SWOT Analysis (Gemini AI)
- Investment Thesis
- Risk Analysis
- Professional PDF Report
- Revenue & Net Income Charts
- 5-Year Stock Price Chart
- Bank-Specific Valuation Logic
